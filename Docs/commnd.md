# コマンド備忘録
## 記載目的
業務で使用していないため、コマンドを忘れがちである現状。
エラー検出されるたびにコマンドを調べる二度手間を防ぐために記載する。

## コマンド
### 初期化
```
go mod init モジュール名
```
新しいGoモジュールを初期化。
最初にgo.modを作るときに使う
### 実行
```terminal
go run main.go
```
一時的にビルドして実行。
複数ファイルの場合は、
```terminal
go run .
```
### 依存関係の整理
```terminal
go mod tidy
```
go.mod, go.sumの依存関係を整理
不要な依存は削除される。必要なもののみが自動追加

### 外部モジュールの取得
```terminal
go get パッケージ名
```
外部モジュールを取得して依存関係に追加
### ビルド
```terminal
go build
```
実行ファイルをビルド。コンパイルのみを行う。
